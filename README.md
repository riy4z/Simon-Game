# Simon Game

A classic Simon game created for fun and to try out jQuery. Test your memory skills and challenge yourself to remember the color sequence!

## Features

- Random color sequence generation
- User input sequence matching
- Score tracking

## Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/yourusername/simon-game.git
    ```

2. Navigate to the project directory:
    ```bash
    cd simon-game
    ```

3. Open `index.html` in your web browser to start playing.

## Usage

1. Click the colors in the sequence shown to you.
2. Follow the sequence as long as you can.
3. The game will reset if you make a mistake.
